Android开发日记（十二）—— Android 6.0权限管理

发表于 2016-11-02   |   分类于 Android   |     |   阅读次数
Android6.0 发布之后，Android 的权限系统被重新设计。在 23 之前 App 的权限只会在用户安装的时候询问一次，App一旦安装后就可以使用所有的权限了，而从 23 之后，App 可以直接安装，App 只有在运行时需要使用某些权限时才会向用户询问是否授权，此时系统会弹出一个对话框让用户选择确认或者取消授权，同时用户也可以在设置页面对每个 App 的权限进行管理。

2015回顾和2016展望

发表于 2016-01-20   |   分类于 生活   |     |   阅读次数

2015回顾：

这一年可能是大学4年里面过的最充实的一年了吧，前半年，从宁波的实习公司辞职，当时离开的时候还是很不舍的，毕竟从大三开始就在那边实习了，而且同一公司的学长也对我们不错。之所以离开是因为想要外面多学点新的技术，这样对自身的发展更有好处，所以下半年来到了杭州。
杭州是六月份过来的，当时被同学拉着进入到现在这家公司（然而我同学已经从这家公司离职了），初创的企业风险摆在那，而我也不幸的中了。虽说公司出现了问题，但自己在Androd这块学到了很多，由于Android这块一直是我在负责，有幸经历了一款App的从无到有，又从有到结束（估计）这样的宝贵经验。从刚开始连屏幕适配都不熟悉到app的连续n个版本迭代;从乱成一团的app代码到现在的简洁框架设计;高速的迭代开发给了我丰富的经历，不过也严重消耗了我的身心，希望接下来能够两边兼顾一起齐头并进。

2015年算是对我而言比较重要的一年吧，这一年从大学毕业了，开始在社会里打滚了~也算是能让人接受的一年，遗憾的是没有找到一个好的团队一起奋斗！

2016展望：

  1. 做一款个人APP并上架
  2. 学习python
  3. 好好的出去玩一趟，一个人也行
  4. 能够找到一个好的团队一起努力
  5. 每周至少要锻炼3-5个小时
  6. 多跟家里联系
