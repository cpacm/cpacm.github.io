2016回顾和2017展望

发表于 2017-01-02   |   分类于 生活   |     |   阅读次数

2016回顾

再过几天，2016年就过去了，在作总结前先看看去年的展望在今年到底实现了多少。

1、做一款个人 APP 并上架

在10月底的时候，总算把 Beats 初稿完成，并在 Github 上开源，断断续续的花了4个月时间。整个应用尽量遵循了 Google 的 Material Design ，并第一次使用 MVP + Rxjava 的模式实现整个应用，有遇到坑也有很大的收获。最后之所以没有把应用上架到商店去，是因为整个应用是基于第三方服务器的 api 实现的，我并不能保证服务的稳定性，所以选择放在 Github 上开源而不是选择商店发布。

其实在写这款完整的个人 APP 之前，还在6月份的时候开源了 SimpleSlider，一个用来实现高度定制化的无限轮播器，一劳永逸地解决应用里面设置轮播图的麻烦。除此之外还附带两个小型的开源项目：
（1）MoneyTextView：解决文字和数字糅合在一起样式多样性的问题。
（2）BeadBattle：一款简单的滑珠游戏，主要目的是熟悉应用中 Animation 的使用。

2、学习python

还在不断学习中，当前只是过了一遍 python 的基础教程，离实用还存在一段很大的距离。正所谓学习一门语言最好的方法就是不断的练习，所以现在刚在阿里云上搭建完 flask，准备用 python 作为服务器语言开一个新的项目。

3、好好的出去玩一趟，一个人也行

趁着杭州G20期间，和原宿舍的室友们去了趟苏州，然后转战上海迪士尼，5天时间的旅游。

4、能够找到一个好的团队一起努力

果然还是要看机遇吗？╮(￣▽￣”)╭
好吧，现在公司的氛围还是不错的，比起上一家公司的话。但总觉得少了什么，对产品的追求？对技术的热忱？可能都有吧。

5、每周至少要锻炼3-5个小时

.(;ˊД ˋ)果然对于像我这样不喜欢运动的人太难了吧，先蹲墙角去好好的反思一下吧。

6、多跟家里联系

马马虎虎吧，不做过多评价。

总结

工作上，换了家公司，相比上一家来说比较轻松（上一家创业公司在我走后没多久就已经倒闭，至于为什么会倒闭，以自己的眼光来看，主要还是因为资金的过于消耗以及技术人才的不足吧，可能也跟boss的战略布局有关，特别在最后的地推环节没有得到很好的收益），所以我也有时间鼓捣一些额外的技术。

生活上，基本上没有什么大的变动，只不过入了ps4的坑，买了不少游戏，然而并没有很多的时间去玩o(╯□╰)o
博客上的写作基本上被我荒废了，除去寥寥几篇的文章就没有其他的产出了，有时候都想好要写的素材了都懒得下手去写（很多文章都随手写在有道笔记上，然而却懒得整理发布出来），这一点明年一定要改正，坚持才能收获成功。

  • 2016 Github上开源了几个自认为不错的项目
  • 2016 看了不少的电影，豆瓣上8分的电影基本上都去看了
  • 2016 书看的有点少了，精读了《Java编程思想》
  • 2016 入了PS4的坑，玩了不少游戏
  • 2016 开始学习python
  • 2016 依旧喜欢着二次元
  • ……

明年将是自己的本命年，自出生来马上要两个轮回了。自强不息，学而不怠，准备好迎接2017年吗？

2017展望

  1. 每周一篇文章，学会写作，持续输出。
  2. 开发出一款独立的产品（包括微信公众号，服务器，Android客户端和web端）。
  3. 每周坚持锻炼3个小时。
  4. 培养自己的 LeaderShip 。
  5. 学习营销的技巧。
  6. 例行一次的出去旅游。
  7. 学习英语。
  8. 细读《编程艺术》，学习算法。

明年基本上就是围绕着要开发的独立产品展开，想法是美好的，但将想法付诸于行动是艰难的，诸君共勉之。
